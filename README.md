# wfc
Wave Function Collapse in C++20

This requires the C++ version `pcg_random`.
Also, make sure you recurse submodules when cloning.

Example:
```
$ ./wfc land_sea_coast.txt 20 10
0: ....................
1: ....................
2: ......C.............
3: .....C~C............
4: ....C~~~C...........
5: ...C~~~~~C..C.......
6: ..C~~~~~~~CC~C......
7: .C~~~~~~~~~~~~C.....
8: C~~~~~~~~~~~~~~CCC..
9: ~~~~~~~~~~~~~~~~~~CC
```
